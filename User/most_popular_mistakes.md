# Most Popular Mistakes

## Status początkowy

Brak jeszcze zarejestrowanych, powtarzalnych błędów językowych użytkownika.

## Zarejestrowane wzorce

- obszar: use-of-english
	błąd: dependent prepositions and fixed chunks
	poprawna forma: `aware of`, `kind of`, `sensitive to`, `contribute to`
	krótka uwaga dydaktyczna: w open cloze trzeba rozpoznawać całe połączenia, nie pojedyncze słowa
	data ostatniego wystąpienia: 2026-04-12

- obszar: use-of-english
	błąd: incomplete transformation structures in key word transformation
	poprawna forma: `is thought to be`, `didn't need to rewrite`, `have been learning`, `needn't have bought`
	krótka uwaga dydaktyczna: w KWT trzeba pilnować pełnej struktury gramatycznej, czasu i aspektu, nie tylko ogólnego sensu
	data ostatniego wystąpienia: 2026-04-12

- obszar: reading
	błąd: weak discourse matching in gapped text
	poprawna forma: contrast, result and accountability links must match the surrounding sentences
	krótka uwaga dydaktyczna: szukaj relacji logicznej między zdaniami, nie tylko zgodności tematu
	data ostatniego wystąpienia: 2026-04-12

## Jak aktualizować

Zapisuj tylko błędy, które:

- pojawiły się więcej niż raz,
- mają znaczenie dla wyniku egzaminacyjnego,
- wpływają na komunikację lub jakość odpowiedzi,
- powinny wracać w kolejnych ćwiczeniach.

## Format wpisu

- obszar: grammar / vocabulary / writing / speaking / use-of-english,
- błąd,
- poprawna forma,
- krótka uwaga dydaktyczna,
- data ostatniego wystąpienia.
