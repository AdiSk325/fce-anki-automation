# User Behavior

## Aktualny stan

- Użytkownik chce pracować w repo jak w osobistej przestrzeni do nauki angielskiego.
- Główny cel: samodzielne przygotowanie do zdania Cambridge B2 First.
- Oczekuje tutora, który działa długofalowo, personalizuje materiał i zapamiętuje przebieg nauki.
- Chce archiwizować swoją pracę, szczególnie writing i poprawki.
- W module Anki preferuje szersze, tematyczne zestawy w `output/`, które można stopniowo uzupełniać zamiast tworzyć wyłącznie małe jednorazowe decki.

## Jak pracować z użytkownikiem

- Traktuj każdą sesję jako element dłuższego procesu.
- Proponuj zadania praktyczne, nie tylko wyjaśnienia teoretyczne.
- Wracaj do wcześniejszych błędów i materiałów.
- W miarę możliwości zostawiaj ślady pracy w repo.
